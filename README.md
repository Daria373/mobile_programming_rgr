# Моя Бібліотека

Android-додаток для перегляду популярних книг та новинок з Google Books API.  
Дозволяє переглядати детальну інформацію про книги, додавати їх до списку «Хочу прочитати» та працювати в офлайн-режимі завдяки локальному кешуванню.

**Розроблено в рамках розрахунково-графічної роботи з дисципліни «Програмування мобільних пристроїв»**  

## Основні функції

- **Популярні книги** – список актуальних книг за запитом `subject:fiction`
- **Новинки** – книги 2024–2025 років (бестселери, фентезі, трилери, романи)
- **Хочу прочитати** – особистий список бажаних книг з можливістю додавання/видалення
- **Детальний перегляд** – повна інформація про книгу (обкладинка, опис, автори, рейтинг, кількість сторінок)
- **Повноекранний перегляд обкладинки** – клік на зображення відкриває його на весь екран
- **Офлайн-режим** – всі завантажені книги доступні без інтернету
- **Оновлення pull-to-refresh** – зручне оновлення списків

## Використані технології

- **Kotlin** – основна мова розробки
- **Android Jetpack**:
  - Navigation Component
  - ViewModel + LiveData/StateFlow
  - Room (локальна база даних)
- **Retrofit + Gson** – мережеві запити до Google Books API
- **Glide** – завантаження та кешування зображень
- **Material Design 3** – сучасний інтерфейс (MaterialComponents)
- **Coroutines + Flow** – асинхронна обробка даних
- **SwipeRefreshLayout** – оновлення списків жестом

## Архітектура

Додаток побудовано за принципом **MVVM** (Model-View-ViewModel).

## Встановлення та запуск

1. Клонуйте репозиторій

2. Відкрийте проект у **Android Studio**

3. Синхронізуйте проект з Gradle (Android Studio зробить це автоматично)

4. Запустіть на емуляторі або реальному пристрої (мінімум API 21)

> Додаток використовує інтернет для першого завантаження даних. Після цього працює офлайн.

## API

Використовується відкритий **Google Books API** (без ключа):
- База URL: `https://www.googleapis.com/books/v1/`
- Документація: https://developers.google.com/books/docs/v1/using
